Extends Advanced Custom Fields with a new field type in order to provide a taxonomy relationship

== Description ==

This plugin extends Advanced Custom Fields with a new field type in order to provide a taxonomy relationship

= Compatibility =

This ACF field type is compatible with:
* ACF 5
* ACF 4

== Installation ==

1. Copy the `acf-taxonomy-relationship` folder into your `wp-content/plugins` folder
2. Activate the Taxonomy Relationship plugin via the plugins admin page
3. Create a new field via ACF and select the Taxonomy Relationship type
4. Read the description above for usage instructions
